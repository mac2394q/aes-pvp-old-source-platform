IT Standards Spreadsheet Standard


Note: Spreadsheets identified as being under purview of Sarbanes Oxley 404 are subject to separate guidelines which might be more stringent than those outlined below. Please contact your SOX 404 Champion or email SOX404@Praxair.com for more details.

Applicability: These standards apply when spreadsheets are:

- Used to store or process information which is then passed on to other transactional systems;

- Have formulas and macros built into them, and the results of such processing of the formulas are either used to enter data into other transactional or reporting system, or are heavily relied upon to drive business decision-making

In either of the above cases the user should inform the IT application manager assigned to his department or business regarding the use of the same. The IT application manager will then provide support to the user in compliance with the standards and if required might undertake an analysis to determine if the needs are better served by an application

Documentation and Change Management

- All macros and formulas used in the spreadsheet should be documented by the user and stored in a departmental folder and the IT application manager needs to be aware of the storage location of the documentation.
- If there are changes in the macro design, the user must ensure version control is followed through numbering and also ensure documentation is updated.

Storage and Backups

- It is preferred that spreadsheets are stored on secured departmental network drives (US- X drives) or personal drives (US - Y drives) to facilitate backups. If the information is highly confidential or is critical to financial statement integrity and accuracy or for decision-making, passwords may be assigned, but the passwords need to be known by the users manager or another backup person.
- If the user prefers to maintain the file on his workstation, the user has to take responsibility for backup by storing a copy of his personal or departmental drive, unless IT has installed software to backup data on PCs (This has been implemented in the US).
